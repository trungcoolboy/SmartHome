# Smart Home

Khoi tao khung du an cho he thong `Smart Home + Futuristic Aqua`.

## Cau truc

- `docs/`: tai lieu kien truc va dinh huong giao dien
- `frontend/`: skeleton dashboard web

## Muc tieu giai doan nay

- chot khung thong tin dashboard
- tao style token theo huong iOS/HomeKit
- dung layout overview de sau nay tach tung cum

## Dashboard sections

- `Overview`
- `Smart Home`
- `Aqua Core`
- `Aqua Motion`
- `Aqua Lighting`
- `Logs & Alerts`

## Ghi chu

Skeleton frontend duoc scaffold theo `React + Vite`, nhung chua cai dependencies trong buoc nay.
